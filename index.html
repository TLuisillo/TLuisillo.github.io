Conexion SSH

Este comando nos sirve para acceder de un servidor Linux a otro servidor Linux, para esto digitamos el comando:
- SINTAXIS: ssh -p 22 user209@hostname.com

XARGS

Es un comando en Unix y en la mayoría de los sistemas operativos tipo Unix utilizados para construir y ejecutar comandos a partir de la entrada estándar.

ls -a

Este comando nos permite ver los archivos ocultos de un directorio.
(Tambien podriamos usar el comando: find . )

Type

Type se utiliza para mostrar información sobre el tipo de comando.
- Podemos usar el comando find y el parametro -type f para filtrar por files, de igual manera el parametro -type d para buscar por directories
Con el parametro -name para buscar por nombre

Cat

El comando cat significa «catenate». Básicamente lo que hace el comando cat es leer datos de archivos y mostrar sus contenidos.

Env

Es un comando del sistema operativo Unix. Es usado para mostrar una lista de las variables de entorno y para ejecutar otro programa con las variables de entorno.

Hostname

Usando el comando hostname -I podemos ver la direccion ip de nuestra red.

Echo

Usando el comando echo mostramos en consola alguna cadena.

Ping

Con el comando ping podemos verificar que tenemos conexion a un servidor, a la vez que nos muestra el tiempo de respuesta.

Chmod

El comando chmod se utiliza para cambiar los permisos de lectura, escritura y ejecucion de algun archivo.

Git Clone

El comando git clone no sirve para clonar o descargar algun repositorio de github.

CSS

Justify Content

Tipos de texto justificado con CSS

Modelo de Caja

El modelo de caja CSS es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual.

Display Block/Inline

block: hace que el comportamiento del elemento sea como un bloque. inline: el elemento se renderizará en línea con otros elementos. inline-block: el elemento tendrá un comportamiento mezcla entre los dos anteriores.

Hover

La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor (puntero del mouse).

Box Shadow

La propiedad CSS box-shadow añade efectos de sombra alrededor del marco de un elemento. Se pueden definir múltiples efectos separados por comas. La caja de la sombra se describe por los desplazamientos en X e Y, los radios de desenfoque y dispersión, y el color relativos al elemento.

Selectores Css

El selector Css es el nexo de unión entre la hoja de estilos y los documentos a los que se aplique dicha hoja.
El selector Css es el identificador en la hoja de estilos del elemento o elementos del DOM a los que se aplicarán las declaraciones contenidas en la regla de la que forma parte dicho selector.

Bootstrap 5

Bootsrap5

Bootstrap es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web.
-Documentacion Oficial: https://getbootstrap.com/

Colores (fondo)

Bootstrap tiene en un amplio sistema de colores que le dan estilos a los componentes. Esto permite una personalización y ampliación más completa para cualquier proyecto.

Container

Los contenedores son usados para rellenar el contenido dentro de ellos, y existen dos clases de contenedores disponibles. . container: provee un contenedor responsive de ancho fijo. container-fluid: provee un contenedor responsive que utilizará todo el ancho de la pantalla.

Float

Estas clases de utilidad hacen flotar un elemento a la izquierda o a la derecha, o desactivan la flotación, basándose en el tamaño actual de la ventana gráfica utilizando la propiedad CSS float.

Cards

Una tarjeta es un contenedor de contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido, colores de fondo contextuales y potentes opciones de visualización.

Carousel

El carrusel es un pase de diapositivas para recorrer una serie de contenidos, construido con transformaciones CSS 3D y un poco de JavaScript. Funciona con una serie de imágenes, texto o marcas personalizadas.

Button

Bootstrap tiene estilos de botones personalizados para acciones en formularios, diálogos y más, con soporte para múltiples tamaños, estados y más...

Navbar

Una barra de navegación es una sección de una interfaz gráfica de usuario destinada a ayudar a los visitantes a acceder a la información.

BreakPoints

BreakPoints son anchos personalizables que determinan cómo se comporta el diseño responsivo en función del tamaño del dispositivo o de la ventana gráfica en Bootstrap.
